El objetivo de este ejercicio es lograr, mediante jQuery, que un grupo de DIV's se comporten como botones o pestañas, que, al hacerles click muestren determinados elementos HTML.

Se sugiere seguir estos pasos en orden para solucionar el ejercicio.

Es importante realizar un paso a la vez, verificando que el mismo funcione correctamente antes de continuar con el siguiente.

1 - Asegurarse de que jQuery esté incluido en la página web antes de querer hacer uso de la función $().

2 - Inicializar jQuery a través de una función ini() disparada por el evento ready() del objeto document, para que los procedimientos se ejecuten una vez que todos los elementos HTML de la página estén disponibles. Verificar que dicha función sea llamada por jQuery cuando la página termine de cargar, definiendo esto con el método ready().

3 - Agregar dentro de las funciones cargarHeladeras(), cargarEquipos(), cargarMicroondas() y cargarNotebooks() vacías, una línea de código que consista en una función alert() con un texto descriptivo que funcionará a modo de prueba. Por ejemplo, dentro de la función cargarHeladeras(), agregar un alert con el texto "Se cargan las heladeras". Realizar lo mismo para las cuatro funciones nombradas, cada una con un texto descriptivo distinto. Verificar que se muestre cada ventana alert al dispararse el evento click de cada uno de los 4 DIV's.

4 - Dentro de la función ini(), buscar con jQuery los elementos con ID "botonHeladeras", "botonEquipos", "botonMicroondas" y "botonNotebooks" y guardar dichas referencias en variables con el mismo nombre. Posteriormente, dentro de la misma función ini() especificarle, mediante el método click(), a cada una de estas variables, que cuando se dispare el evento click debe ser llamada una función. Por ejemplo, se debe especificar para la variable botonHeladeras que cuando se dispare el evento click sobre ella, la función cargarHeladeras debe ser llamada. Si todo funciona hasta aquí, cuando se haga un click sobre el div con ID "botonHeladeras", se debería mostrar un alert con el texto descriptivo elegido en el paso 3. Si esto funciona adecuadamente, realizar lo mismo para las otras variables. Una vez finalizado esto, si al hacer click en cada div (botonHeladeras, botonEquipos, botonMicroondas y botonNotebooks) se visualizan los textos con la ventana alert correspondiente, seguir con el siguiente paso.

5 - Crear una variable global con el nombre "contenedor", sin contenido. Dentro de la función ini buscar, mediante la función $(), el elemento con ID "contenedor, y guardar la referencia hacia ese objeto en la variable contenedor. Modificar la función cargarHeladeras para que en lugar de mostrarse un mensaje descriptivo con un alert, el mensaje descriptivo sea mostrado dentro del elemento contenedor, a través del método .html(). Si esto se realiza de manera correcta, al hacerse click en el DIV con ID "botonHeladeras", debería visualizarse el texto especificado dentro del DIV contenedor. Si esto funciona correctamente, realizar el mismo procedimiento para las funciones cargarEquipos, cargarMicroondas y cargarNotebooks.

6 - Por último, definir 4 variables globales de nombre listadoHeladeras, listadoEquipos, listadoMicroondas y listadoNotebooks, sin contenido. Luego, dentro de la función ini() buscar los elementos HTML con ID con esos mismos nombres, y guardar cada referencia en la variable correspondiente. Por último modificar la función cargarHeladeras para que, en lugar de mostrar un texto fijo dentro del DIV contenedor, se muestre en su interior el html que existe actualmente en el objeto listadoHeladeras, recordando que el método html() permite "leer" el contenido HTML de un elemento cuando se ejecuta el método con paréntesis vacíos, o "escribir" contenido HTML cuando sus paréntesis tienen contenido. Si esto se realizó correctamente, el listado de heladeras ya debería verse correctamente. Realizar este mismo paso para los otros tres.

7 - Paso adicional: Al final de la función ini(), ejecutar la función cargarHeladeras(), para que ni bien cargue la página dicho listado ya se muestre.
